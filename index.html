What is TOON?

Token-Oriented Object Notation (TOON) is a compact, line-oriented serialization format designed specifically for LLM interactions. It preserves the JSON data model while cutting token usage by 30-60%.

How It Works

JSON repeats field names for every single entry. TOON declares field names once at the top, then lists values in clean rows.

items[2]{id,name,role}:
  1,Alice,admin
  2,Bob,user

Token Savings

The same data in JSON costs ~30-60% more tokens.

  • Tabular (uniform objects): 40-60% savings
  • Semi-structured: 15-30% savings
  • Deeply nested: 30-40% savings

Key Benefits

  • Lower API bills — fewer tokens = less cost
  • Faster responses — fewer tokens processed
  • Better accuracy — structured format reduces ambiguity
  • Lossless round-trip — convert between JSON and TOON freely
  • Open standard — github.com/toon-format/spec
